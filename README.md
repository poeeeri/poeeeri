## Featured Projects

### [Chebubrya Memes Bot](https://github.com/poeeeri/chebubrya-memes-bot)

Telegram bot for semantic meme retrieval in group chats.
Combines embeddings, vector search, local retrieval/reranking models, PostgreSQL feedback logging, Docker deployment, and automated security checks.

**Stack:** Python, PyTorch, sentence-transformers, CrossEncoder, ChromaDB, PostgreSQL, aiogram, Docker, GitHub Actions

---

### [Mindly Memory Agent](https://github.com/poeeeri/Mindly-Memory-Agent)

Conversational LLM agent with persistent long-term memory.
Built with FastAPI, LangGraph, OpenRouter, MemPalace/ChromaDB, PostgreSQL, Docker, and a React/Vite WebUI.

**Stack:** Python, FastAPI, LangGraph, LLM, OpenRouter, MemPalace, ChromaDB, PostgreSQL, Docker

---

### [License Detector](https://github.com/poeeeri/license-detector)

Vehicle license plate detection and OCR system.
Includes custom dataset preparation, YOLOv8 training, TrOCR/PaddleOCR experiments, video/camera inference, and ClearML tracking.

**Stack:** YOLOv8, TrOCR, Hugging Face, PaddleOCR, ClearML, CV

---

### [Banking Support](https://github.com/poeeeri/Banking-Support)

NLP pipeline for banking support automation.
Generated synthetic dialogues and trained a RuBERT-based intent classifier with experiment tracking in ClearML.

**Stack:** NLP, RuBERT, PyTorch, ClearML
